Drop your converted duck model here as: duck_baby.glb

Convert Duck_Baby.fbx -> duck_baby.glb using Blender (free):
  1. Open Blender, delete the default cube
  2. File > Import > FBX(.fbx) -> select Duck_Baby.fbx
  3. File > Export > glTF 2.0 (.glb/.gltf) -> Format: glTF Binary (.glb)
  4. Save as duck_baby.glb in this folder

Until this file exists, all three clients automatically fall back to a
placeholder low-poly duck built in Three.js, so the game stays fully playable.
